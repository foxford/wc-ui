# webcomponents-ui components

## Tabs

### Tabs with generated bars
```html
  <tab-list selected="Tab 1">
    <tab-item title="Tab 1">Tab 1 content</tab-item>
    <tab-item title="Tab 2">Tab 2 content</tab-item>
  </tab-list>
```

## Headless tabs
```html
  <tab-list selected="Tab 2" headless>
    <tab-item title="Tab 1">Tab 1 content</tab-item>
    <tab-item title="Tab 2">Tab 2 content</tab-item>
  </tab-list>
```
